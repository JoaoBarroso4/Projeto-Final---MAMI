Calculadora Gráfica - Projeto Final de MAMI

  • Este programa permite apenas a plotagem de gráficos de funções com uma única variável, sendo esta x.
  • Para plotar o gráfico da função, a notação f(x) ou equivalente deve ser substituída por apenas y.
  • É possível fazer uso dos templates de funções matemáticas (logaritmo, divisão, radiciação, etc.) utilizando o teclado de botões abaixo de onde a função é dada como entrada.

Para dar zoom, utilize o scroll do mouse ou o gesto de pinça.
